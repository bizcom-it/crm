Kalender und Ressourcenplaner

Mehrere Reiter am oberen Rand steuern, welcher Inhalt gezeigt wird.

Alle, für den Benutzer freigegebenen Kalendereinträge, werden hier angezeigt.

Unter Ressourcen werden eingestellte Ressourcen dargestellt und gebucht.

Mit Hilfe des dritten Reiters, werden nur die Termine des aktuellen Benutzers angezeigt.

Mindestens zwei weitere Reiter werden noch angezeigt. Diese blenden Reiter anderer Benutzer oder Kategorien ein oder aus.
Dort werden die zugeordneten Termine angezeigt.

[Kalendereintrag]
Wählen sie den gewünschten Zeitraum durch ziehen mit der Maus aus. Es wird ein Dialogfenster geöffnet.
Vergeben sie einen aussagefähigen Titel ein. Prüfen und ändern sie ggf. den Zeitraum und legen sie einen Priorität fest.
Der Termin kann einer Kategorie oder einem anderen Benutzer zugeordnet werden. Auch können sie festlegen, wer den Termin sehen kann (Alle, nur der aktuelle Benutzer oder eine angelegte Gruppe).
Im Feld 'Kunde / Location' können sie einen Kunden, Lieferanten oder eine Person zuordnen. Geben sie min. zwei Zeichen ein, so werden bereits alle Treffer angezeigt. Sie können aber auch einen Ort eingeben.
Ein Termin kann auch wiederholt werden. Als letztes kann noch eine Langbeschreibung eingegeben werden.
Angelegte Termine können mit der Maus verschoben werden. Eben so kann die Dauer verändert werden.
[Ressourcen]
Ressourcen werden duch einen autorisierten Benutzer eingestellt.
Nur dieser Benutzer kann gebuchte Ressourcen genehmigen oder frei geben.
Ressourcen werden nicht im normalen Kalender angezeigt.

Die Eingabe eines Ressourceneintrags, ist mit dem eines Kalendereintrages nahezu identisch. Einziger Unterschied ist die erste Zeile. Nach Eingabe der ersten zwei Zeichen wird eine passende Ressource gesucht (__ findet alle Ressourcen). Es kann nach Einträgen zu einer Ressourcengruppe gesucht werden oder nach der Ressourcengruppe.
Gruppen sind z.B. 'Dozent' oder 'PKW'. Sind weder Gruppennamen, noch zugeordnete Ressourcen bekannt, können alle Einträge abgerufen werden in dem 2 Prozentzeichen (%) oder Unterstriche (_) eingegeben werden.
