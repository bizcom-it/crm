Etiketten

Wird das Etikettenmodul aus den Stammdaten oder aus der Rechnung aufgerufen, so wird in einem Pop-Up nur ein Eingabefeld angeboten. Je nach Konfiguration des Etiketts wird es ausgedruckt.

Wird das Modul aus dem Menü aufgerufen, so werden verschiedenen Eingabefelder im Hauptfenster angeboten. Sie haben so die Möglichkeit, Etiketten zu erstellen, für die keine Adressdaten gespeichert sind.

Im Auswahlmenü werden die vorher definierten Etiketten angeboten.
Die Anzahl und Position der Radiobutton stellen die Druckposition für das Etikett auf einem Blatt dar.
