Auftragschance

Hier die Chancen und der Status eines Projektes mit einem Kunden bewertet werden. Auch hinterlegen sie den nächsten Schritt.

Wird die Seite über die Firmenstammdaten aufgerufen, so werden alle Auftragschancen gesucht und ggf. in einer Liste aufgeführt. Ein einzelner Treffer kommt sofort zur Anzeige. Ohne Treffer wird eine Eingabemaske angeboten.

=[Suche]=
Geben sie einige Buchstaben für eine Firma ein. Bis zu 11 Treffer werden automatisch in einer Dropdown-Liste angezeigt. Je mehr Buchstaben eingegeben werden, desto besser werden die Ergebnisse. Wählen sie die gewünschte Firma aus.
Jetzt können sie noch weitere Suchbegriffe eingeben. Die Felder werden "UND"-Verknüpft.

=[Eingabe]=
Wird eine bestehende Auftragschance aufgerufen, so werden sind Daten einfach nur angezeigt. Erst ein Klick auf einen Wert öffnet eine Eingabezeile. Erst wenn sich ein Wert ändert, wird auch der Knopf zum Sichern angezeigt.
Die Dropdownliste Auftrag enthält alle offenen Aufträge dieses Kunden.
