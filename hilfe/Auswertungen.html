Verkauf Auswertungen

Wird nur das "Von-Datum" eingegeben, so wird ab diesem Zeitpunkt bis zum aktuellen Datum ausgewertet.
Wird nur das "Bis-Datum" eingegeben, so wird von Anfang bis zu diesem Datum ausgewertet.
Bei den Monatsumsätzen, wird beim "Datum" der ganze ausgewählte Monat benutzt.
Bei den Tagesumsätzen wird der Monat und das Jahr im "Von-Datum" ausgewertet.

Bei einigen Berichten kann die Suche noch weiter eingeschränkt werden z.B. "Umsatz nach Wochentag".
Es kann genau gesucht werden, aber auch mit Jokerzeichen. % = beliebige (0-n) Zeichen _ = genau ein Zeichen. Diese Zeichen können in beliebiger Folge verwendet werden:
Artikelnummer: D1%    sucht alle Artikelnummern die mit "D1" beginnen

Bei anderen Berichten kann vor den Suchbegriff ein Logikzeichen verwendet werden, Joker sind hier nicht erlaubt:
= sucht auf Gleichheit
< sucht kleiner als
> sucht größer als
Monatsbericht - Umsatz: >500    Umsatz größer 500€, bezieht sich nicht auf einzelne Rechnungen, sondern auf die Summe.
