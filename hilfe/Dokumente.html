Dokumente

Wird die Dokumentenverwaltung aus dem Reiter 'Dokumente' den Ardressstammdaten aufgerufen, so werden zunächst die gekürzten Firmendaten angezeigt. Daneben bekommen sie zwei Auswahlmenüs für die Erstellung von Dokumenten und Wartungsverträgen.

Die Dokumentenverwaltung ist in zwei Hälften unterteilt. Im linken Fenster wird das aktuelle Verzeichnis ausgegeben.
Wir ein Verzeichnis angeklickt, so wird dahin gewechselt. Klickt man auf eine Datei, so werden in der rechten Hälfte weitere Informationen zu der Datei ausgegeben. Auch sind dort Dateimanipulationen möglich.
Im oberen Bereich der linken Hälfte, werden bis zu 4 Knöpfe angezeigt, mit deren Hilfe es möglich ist:
  • Eine Datei hochladen
  • Ein neues Verzeichnis erstellen
  • Das aktuelle Verzeichnis löschen
  • Das aktuelle Verzeichnis neu einlesen
In der rechten Hälfte wird immer der Knopf 'Filebrowser' angezeigt. Damit lässt sich auch ein Verzeichnis darstellen.
Ist ein Dokument ausgewählt, werden folgende Knöpf angezeigt:
  • download - Die aktuelle Datei zum PC übertragen
  • löschen - Die aktuelle Datei löschen
  • verschieben - Die Datei in ein anderes Verzeichnis verschieben
  • Attribute verarbeiten - Ein Beschreibungstext kann eingegeben werden und die Datei umbenannt werden
  • Datei sperren - Die Datei wird gegen Veränderung geschützt. Kann nur durch den User geändert werden, der die Sperre gesetzt hat.
