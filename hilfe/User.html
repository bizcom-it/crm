Sync
Datensynchronisation mit DAV-Server, z.B. Owncloud
zZ wird nur Adresse unterstützt.
DAV-Serverkomplette URL
Name/Passwortnötig für die Anmeldung
erl. Errorcodeerlaubte Fehlercodes neben 201. Owncloud liefer z.B. 204
SyncstatusWelchen Status bekommen neue Adressen
holen holt die Daten vom Server
Dargestellt werden alle Adressen seit dem letzten Holen in einem neuen Dialog.
Dabei wird an Hand eines Keys oder einer UID versucht die Adresse in der Datenbank zu finden. Bei Erfolg wird neben dem Namen je nach Syncstatus (0,1,2) der Adresse eine Checkbox (2) für update, "no" wenn der Status 1 ist oder ein "?" bei Status 0 angezeigt, Bei nicht gefundenen Adressen wird eine Auswahlbox mit den Tabelle angezeigt, wo die Adresse eingefügt werden soll. Sind sie mit Ihrer Auswahl fertig, so klicken sie auf anwenden. Die selektierten Daten werden dann übernommen.

senden überträgt alle Adressen mit entsprechendem Marker (Synstatus 1 oder 2).
