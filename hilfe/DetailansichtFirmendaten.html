DetailansichtFirmendaten

Hier werden die Firmendaten detaliert angezeigt. Die Information wird über vier Reiter bereitgestellt. Neben den Reitern befindet sich noch ein Aktionsmenü.
[Mitteilungen] erscheinen über dem Hauptmenue. Sie haben je nach Priorität andere Darstellungen der Wichtigkeit.

Firmenstamm
Je nach Benutzereinstellungen werden mehr oder weniger Symbole in dem Feld der Rechnungsmaske angezeigt. Im Drop-Down Menü [Aktionen] werden diese aber auch aufgeführt. Sind in der Wissen-DB entsprechende Einträge, wird auch noch die Auswahl [Hilfe] angeboten.

Das dreigeteilte Anzeigefenster enthält Informationen über die Rechnungsanschrift, darunter weitere Informationen in Reitern und rechts Reiter für verschiedene [Listen], unter anderem einen [Kontaktverlauf].

[Rechnungsanschrift]
In ihr finden sich alle wichtigen Daten auf einen Blick. Ist eine E-Mailadresse eingegeben, so wird diese als Link dargestellt und führt zu dem eingebauten eMail-Client. Eine evtl. vorhandene Web-Adresse wird auch als Link angezeigt und öffnet ein neues Fenster mit der angegebenen URL.

Mit einem Klick auf das Briefsymbol wird ein neues Fenster geöffnet. Dort wird die Adresse als reiner Text dargestellt um sie mit Copy + Paste in eine andere Anwendung übernehmen zu können. Auch kann hier ein Etikettdruck gestartet werden. Der Button Auftrag wechselt in die Maske "Auftrag Erfassen" im Menü "Verkauf" der ERP. Der Knopf Angebot ruft die Maske "Angebot Erfassen" auf. Zwei weitere Symbole verzweigen zu Lieferschein und Rechnung erzeugen. Die kleinen Karte ruft einen eingestellten Mapserver in einem Extrafenster auf.

[weitere Informationen]
Fünf Reiter halten unterschiedliche Informationen bereit. Die bevorzugte Information kann am Benutzer eingestellt werden.
Lieferanschrift
Ist eine Lieferadresse bei den Stammdaten hinterlegt, so wird diese angezeigt, an sonsten die Rechnungsanschrift.
Gibt es weitere Lieferanschriften, so können diese mit den Pfeilen abgerufen werden.

Notizen
Die Stichworte und interne Bemerkungen werden hier ausgegeben.

Variablen
Sind Variablen angelegt, so werden diese hier neben den Firmenspezifischen Inhalten angezeigt.

Finanzinfos
Informationen wie Bankverbindung, Kundentyp und Zahlungskonditionen, Rabatt und Preisgruppe, offene Posten und Aufträge stehen hier bereit.

zusätzliche Infos
Gehört diese Firma einem übergeordneten Konzern an, so wird der Name in den eckigen Klammern angezeigt. Ein Klick darauf ruft die Daten ab. Sind andere Firmen dieser Firma untergeordnet, so wird die Anzahl in den runden Klammern ausgegeben. Wird die Zahl angeklickt, wird eine Übersichtsseiter aufgerufen.

Auf der rechten Seite sind vier Reiter für unterschiedliche Listen:

Kontakt
Unter diesem Reiter werden, wenn vorhanden, die Kontaktereignisse aufgeführt und es wird die Möglichkeit gegeben einen neuen [Verlauf] einzustellen. Ein Klick auf die entsprechende Zeile öffnet einen entsprechenden Dialog. Der zuletz bearbeitete Eintrag wird fett ausgegeben.

Angebot
Alle Angebote dieser Firma werden hier aufgelistet.

Auftrag
Alle Aufträger dieser Firma werden hier aufgelistet.

Rechnung
Alle Rechnungen und Gutschriften dieser Firma werden hier aufgelistet.

Jede Zeile enthält das Datum, die erste Position des Dokumentes, den Nettobetrag und die Nummer. Ein Klick auf einen Eintrag führt direkt zur Bearbeitungsmaske.

Ansprechpartner
Wird dieser Reiter angeklickt, werden zunächst alle Ansprechpartner der Firma angezeigt. Ein Klick auf die gewünschte Person stellt dann diese ein. Sollte es zu der Firma nur einen Kontakt geben, wird dieser automatisch angezeigt. Das zweiteilige Anzeigefenster enthält im linken Teil die [Daten] einer Kontaktperson, im rechten Teil erscheint wieder ein [Kontaktverlauf] mit dieser Person.
Unterhalb der linken Anzeige gibt es wieder einige Knöpfe um den Eintrag zu [bearbeiten], neue Kontaktdaten [einzugeben] oder eine Kontaktperson aus einer [Personensuche] zu zuteilen.

Umsätze
Ein erster Klick auf den Reiter Umsätze stellt die summierten Umsatzdaten der letzten 12 Monate als tabellarische Liste und als Grafik dar. Mit den Links oberhalb der Grafik können andere Jahre abgerufen werden. Die Grafik stellt die Umsatzdaten logarithmisch dar, mit einem weiteren Klick auf den Reiter Umsätze kann in eine lineare Darstellung umgeschaltet werden und wieder zurrück. Dargestellt werden je Monat: - Gesamtumsatz - durchschnittlicher Umsatz - Gutschriften - Angebote Mit einem Klick auf einen Tabelleneintrag, wird in die Monatsansicht umgeschaltet. Jetzt sind alle Rechnungen und Angebote einzeln sichtbar. Die einzelnen Rechnungspositionen können in einem neuen Fensten durch anklicken dargestellt werden.

Der Link Auftragschancen ruft die eingestellten [Auftragschancen] des Kunden auf.


Dokumente
Ein kleiner [Dokumentenmanager], der direkt im Kundenverzeichnis startet.
Zusätzlich kann hier aus einer [Dokumentvorlagen] ein neues Dokument generiert werden. Mit einem Klick auf das gewünschte Dokument kommen Sie in die [Variablenansicht]. Verschiedene Felder im Dokument werden mit diesen Variablen ersetzt. Dabei können verschiedene Felder bereits mit den Daten einer Firma oder Kontaktperson ausgefüllt sein. Die Felder für Datum und Zeit beziehen sich auf die Serverzeit. Sind alle Felder zu Ihrer Zufriedenheit ausgefüllt drücken Sie einfach nur "Erstellen". Damit wird das eigentliche Dokument erzeugt und zum Speichern angeboten. Gleichzeit wird das Dokument auch bei der entsprechenden Adresse im [Kontaktverlauf] zugeordnet.

Sind zu der Firma Wartungsverträge hinterlegt, so finden sie diese im entsprechenden Drop-Down Menü.

[Hilfe-Pulldown]
Ist unter der [Wissens-DB] mindestens eine Kategorie für Kundenhilfe eingerichtet, wird ein Pulldown-Menü neben den Reitern angezeigt. Die Auswahl eines Menüpunktes ruft die Wissens-DB mit dem gewünschten Thema in einem extra Fenster auf.

[Drop-Down-Menü]
"bearbeiten" ruft die [Bearbeitungsmaske] auf.
[Zeiterfassung] Arbeitszeiten zu Firmenprojekten erfassen.
"Extra Daten"
"VCard" erzeugt eine Textdatei im VCard 3.0 Stiel.
"Copy to Cookie" Speichert die VCard im Cookie, damit kann ein neuer Eintrag generiert werden.
"QR-Code" erzeugt aus der VCard ein QR-Code und bietet das Bild zum Download an.
"Karteikarte" erzeugt ein OOo-Dokument mit den Firmeninformationen für eine Karteikarte.
