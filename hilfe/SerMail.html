Serien E-Mails versenden

An die ausgewählten Adressen kann eine E-Mail gesendet werden. Zusätzlich an einen angelegten User (CC).
Jede Mail enthält einen Betreff, einen Textkörper und kann einen Anhang haben.
Die Mail kann manuell eingegeben werden oder aus einer schon vorhandenen Vorlage bestehen.
Der Textkörper kann Variablen enthalten. Diese werden dur ein Prozentzeichen "%" eingeklammert ( %ANREDE% ).
Als Variablennamen stehen die gleichen wie bei der Dokumenterstellung zur Verfügung.
