[Auswahlliste]

In der Auswahlliste werden alle auf das Suchkriterium gefundenen Firmen und Personen angezeigt. Dargestellt wird bei Firmen:
Kundennummer; Firmenname; Plz; Ort; Straße; Telefon; E-Mail; Gültigkeit

Bei Personen wird der folgende Satz angezeigt:
Nachname, Vorname; Plz; Ort; Telefon; E-Mail; Firmenzugehörigkeit

Durch das Auswählen von einem Listeneintrag wird in die Detailansicht der [Firmen] bzw. [Personen] gewechselt.
Es werden nicht alle Ergebnisse auf einmal dargestellt werden. Seitenweise können weitere Treffer abgerufen werden.

Unterhalb der Liste gibt es eine Reihe von [Aktionsknöpfen]:

Etikettendruck
Mit dem Etikettendruck bekommt der Benutzer die Möglichkeit für die gesammte Auswahlliste auf einmal die Etiketten zu erstellen.
In dem Pulldownmenü unter "Etikett" können die vormals erstellten Etiketten im ["Etiketten-Editor"] ausgewählt werden. Zusätzlich legen sie den Startpunkt für das erste Etikett fest.


Serienmail
Die eMail-Maske wird aufgerufen. Hier können Sie einen Text eingeben und ein Dokument als Anhang auswählen. Der Text kann einige Variablen (siehe ["Etiketten-Editor"]) enthalten.
Mit 'senden' wird an alle zuvor ausgewählten Empfänger gesendet. Dazu öffnet sich ein Fenster, das erst nach der letzten eMail geschlossen werden darf.

Serienbrief
Hier können sie in drei Feldern noch Werte ausfüllen. Ein Datum, eine Betreffzeile und ein Freitext. Diese werden ebenso wie die Felder der Suchergebnisse in dem noch hochzuladenden Dokument eingefügt.

VCards
Mit dem Suchergebnis kann ein VCard-File erstellt werden, daß zum Download angeboten wird oder für jeden Treffer eine VCard auf dem Server.

autom.Sync
Den Synstatus für die Auswahl setzen.
Nein == 0 == nicht mehr autom. zum Server syncen
nur senden == 1 == nur zum Server senden
beide Richtungen == 2 == vom Server kommede Adresse automatisch aktualisieren
